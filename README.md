中心化的配置管理组件。建立这个项目是为了解决同一公司多个项目多个环境配置不统一、修改麻烦的问题。

### 开始使用

[详细文档](https://liuyuyu.github.io/dictator/)

### 特性

- [x] 中心化配置

- [x] Spring/Spring Boot集成

- [x] web管理界面

- [x] JDK8+

- [x] 自动更新配置
